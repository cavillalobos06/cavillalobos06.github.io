# Rock band page

## Description

The website displays information about a rock band which shows information about tours, venues, dates and a bit of history


It is designed with an adaptable approach, adjusting to both mobile and desktop computers

# Technologies Used
* **HTML5**
* **CSS3**
* **JavaScript**
* **Font Awesome (icons)**
* **Google Fonts** 

This project is a **starter template** for a performance assessment focused on building a responsive landing page for a rock band.

# Hamburger Menu

The project features a hamburger menu using **JavaScript** that is activated on mobile phones to provide a cleaner and more professional interface.
