# Portafolio

**This project is a personal portfolio website developed using HTML, CSS, and JavaScript, where personal information, projects, and an additional section dedicated to pets are presented.**

# Description

### The website allows you to showcase:

* Personal information (About Me)
* Developed projects
* Contact form
* Additional page about pets (PetCare)

It is designed with a responsive approach, 
adapting to different screen sizes.

# Technologies Used
* **HTML5**
* **CSS3**
* **JavaScript**
* **Font Awesome (icons)**
* **Google Fonts**

# Features
## Main Page (`index.html`)
* **About Me** section with dynamic text
* Interactive buttons:
    * `Change text`
    * `Show more`
* Projects section with GitHub links
* Contact form
## 🐾 Pets Page (`mascotas.html`)
* Catalog-style layout
* Informative cards about different pets:
    * Dogs
    * Cats
    * Birds
    * Fish
    * Hamsters 
    * Rabbits
## Design and Styling
* Use of **Flexbox** and **CSS Grid**
* Modern design with effects such as:
    * backdrop-filter
    * box-shadow
    * hover effects
* Fully **responsive** using media queries:
    * 576px
    * 768px
    * 992px
    * 1200px
    * 1400px
# Responsive Design

The website adapts to:

* 📱 Mobile devices
* 📲 Tablets
* 💻 Desktop
* 🖥️ Large screens
